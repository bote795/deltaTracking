#iBeacon
This files are for creating a rasberry pi runing rasberian to a iBeacon.
##Needs

> Rasberry pi
>
> bluetooth 4.0 toogle

##First 

> chmod +x install.sh
>
> ./install.sh

this will take a while
after everything fully installed
(maybe 90 minutes)

##then run  

> chmod +x startbeacon.sh
>
> ./startbeacon.sh

this will run the beacon
